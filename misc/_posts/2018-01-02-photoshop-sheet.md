### 存储为Web格式
``` Ctrl + Alt + Shift + S```
这个是网页设计师经常用到的快捷键之一。

### 裁剪
```
Cmd + Shift + C
Cmd + N
Cmd + V
```

### 导出某个图层为 png
```
* 工具栏选中 Auto-Select 勾上，并选择为 Layer
* Alt + 点击，在右侧 Layers 中对应图层被选中
* Shift + Alt + 点击右侧图层中的眼睛，屏蔽其他图层
* Cmd + Shift + C
* Cmd + N
* Enter
* Cmd + S
* Cmd + Alt + Shift + S
```
